# 🎓 IIM Call Predictor

## Overview
This project predicts the probability of a student receiving an interview call from an **IIM (Indian Institute of Management)** based on their CAT exam percentiles, academic background, work experience, and reservation category.

The model is built using **real IIM CAT cutoff data** sourced from Kaggle (`tejaswis0702/cat-selection-cutoff-iim`), which contains category-wise and college-wise selection cutoffs for 19 IIMs including IIM Ahmedabad, Bangalore, Calcutta, Lucknow, Indore, and more.

## Dataset
The dataset (`iim_admission_predict.csv`) was generated by applying real IIM cutoff rules to simulated applicant profiles. Features include:
- **VARC**: Verbal Ability & Reading Comprehension percentile
- **QA**: Quantitative Aptitude percentile
- **DILR**: Data Interpretation & Logical Reasoning percentile
- **Overall Percentile**: Average of the three sections
- **10th Percentage**: Class 10 academic score
- **12th Percentage**: Class 12 academic score
- **Work Experience**: Work experience in months
- **Category**: General / OBC / SC / ST / EWS / PwD (reservation-based cutoff adjustments applied)
- **Chance of Admit**: Target variable — fraction of IIMs the applicant qualified for (0.0 to 1.0)

## Reservation Logic
Category-wise cutoffs are applied from the real IIM data, so lower CAT cutoffs automatically apply for reserved categories (OBC, SC, ST, EWS, PwD) across all supported IIMs.

## Project Structure
- `build_iim_dataset.py`: Generates the training dataset from real IIM cutoff CSVs.
- `train_model.py`: Trains a Random Forest Regressor and saves `model.pkl` and `features.pkl`.
- `app.py`: Streamlit web app for interactive prediction.
- `run.py`: Command-line interface for quick predictions.
- `iim_admission_predict.csv`: Generated ML training dataset (10,000 records).
- `cat data/`: Real Kaggle IIM CAT cutoff data files.

## How to Run

### Prerequisites
```bash
pip install -r requirements.txt
```

### Step 1: Generate Dataset (Optional — already included)
```bash
python build_iim_dataset.py
```

### Step 2: Train the Model (Optional — model.pkl already saved)
```bash
python train_model.py
```

### Step 3: Launch the Web App
```bash
python -m streamlit run app.py
```
Then open [http://localhost:8501](http://localhost:8501) in your browser.

### Step 4: CLI Mode (Optional)
```bash
python run.py
```

## Results
The Random Forest model learns from real IIM cutoff thresholds, enabling it to differentiate chance of admission across categories (e.g., a General candidate at 92 percentile vs. an SC candidate at 70 percentile). The app provides instant feedback on likelihood of receiving IIM interview calls.
